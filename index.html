Savoir Faire

Intégrer une maquette

A partir d'une maquette, je peux de concevoir l'interface d'une application web responsive et de la rendre dynamique avec Javascript.
Je comprend l'interêt du Mobile First et je suis capable de développer une application de cette manière.

Maitrise d'une librairie / framework JS

Je suis à l'aise avec la librairie React avec laquelle je crée des Single Page Applications performantes.
Pour gérer le State de l'application, j'utilise des librairies tel que Redux ou Recoil.

Développer la partie Back-End(+API)

Pour dévellopper la partie Back d'un application, j'utilise NodeJs ainsi que le framework ExpressJS.
Je peux développer des applications en mode API ReST ou en utilisant une architecture MVC.

Concevoir et consomer une base de données

A partir d'un besoin client, je suis capable de concevoir un Modèle Conceptuel de Donnée à partir duquel je pourrai développer une Base de Données relationnel ou non relationnel.
Pour accéder aux données, je sais écrire manuellement mes requêtes ou je peux mettre en place et utiliser un ORM tel que Sequelize.
